# Code of Conduct

This project follows the [Contributor Covenant, v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

In short: be direct, be honest, be kind. Technical disagreement is welcome; personal attacks, harassment, and discriminatory behavior are not. We moderate accordingly.

## Reporting

Report conduct issues to `forfalsket@proton.me`. All reports are treated confidentially. We respond within seven days.
